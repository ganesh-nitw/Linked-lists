# Linked Lists in C++

This repository contains C++ programs that demonstrate the implementation of
singly linked lists.

## Contents
- Creation of singly linked list
- Create function for node allocation
- Traversal of linked list
- Insertion at a given position
- Deletion at a given position

## Purpose
This repository is created for learning and practicing Data Structures,
especially linked lists, using C++.
